# Experiment 2 - Subtraction

## AIM
To write and execute a program for subtraction using R.

## Program

```r
num1 <- as.integer(readline(prompt = "Enter the first number: "))
num2 <- as.integer(readline(prompt = "Enter the second number: "))

num3 <- num1 - num2

print(num3)
```
